# 🩺 Chronic Kidney Disease Prediction using Machine Learning

A hybrid Machine Learning model that predicts Chronic Kidney Disease (CKD) using ensemble learning techniques.

## 🚀 Features

- Data preprocessing
- Feature engineering
- SMOTE for class balancing
- Ensemble Learning
- Model evaluation
- Interactive Gradio interface

## 🛠️ Technologies Used

- Python
- Scikit-learn
- XGBoost
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Gradio

- Data Cleaning
- Exploratory Data Analysis
